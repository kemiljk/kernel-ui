# @kernelui-lib/elements

Kernel components as native Custom Elements — framework-agnostic, light DOM.

## Install

```bash
npm install @kernelui-lib/elements @kernelui-lib/styles
```

## Usage

```css
@import "@kernelui-lib/styles";
@import "@kernelui-lib/elements/styles.css";
```

```html
<script type="module">
  import "@kernelui-lib/elements";
</script>

<kernel-button variant="primary">Save changes</kernel-button>
```
