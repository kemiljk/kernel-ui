# Security Policy

## Supported versions

| Version | Supported |
| --- | --- |
| 0.1.x | Yes |

## Reporting a vulnerability

Please report security issues privately by opening a [GitHub Security Advisory](https://github.com/kemiljk/kernel-ui/security/advisories/new) or emailing the maintainer through their GitHub profile.

Do not open public issues for undisclosed security vulnerabilities.

We aim to acknowledge reports within 3 business days.
